PHP-NTL

NumberToLetter

NTL is an Open Source PHP librairy multilangage that allows PHP developpers to convert number to letter from 0 to 999,999,999,999 easyly.

Get Started Download

#How to use it?

  • Import the class NumberToLetter.php file to your index page
  • create an Instance of NumberToLetter
    • First param: the number to convert
    • Second param: the langage 'ENG' for English and none or 'FR' for French.The default langage is French.
  • Use the method numberToLetter

#Make many test

  • You can an also make many test using the static:: method NumberToLetter::makeTest($from, $to, $lang)
    • @param from: starting test
    • @param to: ending test
    • @param lang: langage
